## What’s in this repo

- `app/` — Next.js App Router pages and React components
- `data/cards.js` — content for the info cards
- `css/style.css` — main styles
- `css/utils.css` — small utility classes
- `public/assets/` — images and icons used in the layout

## GitHub Pages

The project is prepared for GitHub Pages with a static Next.js export.

1. Push the repository to GitHub.
2. Open repository settings: `Settings` -> `Pages`.
3. In `Build and deployment`, set `Source` to `GitHub Actions`.
4. Push to `main` or `master`.
5. Open the Pages URL shown by the `Deploy to GitHub Pages` workflow.

For this repository name, the site is configured to work under:

```text
https://<your-github-username>.github.io/Site_about_mountains/
```
